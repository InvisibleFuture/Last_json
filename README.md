# Last json
为 xiuno 提供 JSON API 的插件


使用 RESTful 风格, 应理解以下基本概念:
GET    查询操作, 不改变服务器数据的幂等查询, 其参数只有url中的query类型
POST   增量操作, 向指定对象集提交新对象
PATCH  修改操作, 修改指定对象的全部或部分属性字段
DELETE 删除操作, 删除指定对象




用户:

GET /user
获取用户列表(通常只允许管理员无条件查询)

POST /user
新增用户(通常只允许管理员手动创建用户)
